# 1、用例建模概念

## 2、图书馆管理系统的用例模型，画出用例图

### 2.1、基本功能要求：

#### 2.1.1、读者管理

```
读者信息的制定、输入、修改、查询，包括种类、性别、借书数量、借书期限、备注等。
```

#### 2.1.3、书籍管理

```
书籍基本信息制定、输入、修改、查询，包括书籍编号、类别、关键词、备注。
```

#### 2.1.4、借阅管理

```
包括借书、还书，预订书籍、续借、查询书籍、过期处理和书籍丢失后的处理。
```

#### 2.1.5、系统管理

```
包括用户权限管理、数据管理和自动借还机的管理。
```
